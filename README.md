# 农产品碳排放计算平台

一个专业的农产品碳排放计算网站平台，帮助农业生产者科学计算和分析农产品生产过程中的碳足迹。

## 功能特点

- **多种农产品支持**：支持水稻、小麦、玉米、蔬菜、水果、大豆、棉花等主要农产品的碳排放计算
- **全面参数输入**：包括种植面积、产量、化肥使用量、农药使用量、灌溉用水量、机械作业时间等
- **区域差异化**：考虑不同地区的气候和种植条件差异
- **详细结果分析**：提供总碳排放量、单位产量碳排放、排放构成分解等详细数据
- **用户友好界面**：简洁直观的操作界面，易于使用

## 使用方法

1. 在浏览器中打开 `index.html` 文件
2. 选择农产品类型
3. 输入种植面积和产量
4. 填写农业投入品使用情况（化肥、农药、灌溉、机械作业等）
5. 选择种植区域
6. 点击"计算碳排放"按钮
7. 查看详细的计算结果和分析

## 技术架构

- **前端技术**：纯HTML、CSS、JavaScript
- **计算方法**：基于IPCC（政府间气候变化专门委员会）计算方法学

## 碳排放计算说明

本平台采用以下计算方法：

1. **基础种植排放**：根据作物类型和种植面积计算
2. **化肥使用排放**：基于化肥使用量和排放系数
3. **农药使用排放**：基于农药使用量和排放系数
4. **灌溉排放**：基于灌溉用水量和能源消耗
5. **机械作业排放**：基于机械作业时间和燃料消耗

所有计算结果均以CO₂当量（kg CO₂e）表示。

## 注意事项

- 所有输入数据应为实际生产数据
- 计算结果仅供参考，实际碳排放可能因具体条件而异
- 建议结合专业指导进行碳排放管理
